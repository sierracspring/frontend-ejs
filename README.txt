What type of resources can you host with Express?
-Websites, web apps, and API's.

What are the benefits of using a View Engine over 
using Vanilla technologies?
-It starts a strong templating foundation for the coder.


Why does the Express Generator split up the functinality 
into so many files?
-This is done for organization. As projects grow larger,
it becomes increasingly more helpful to be able to jump
to a specific file, rather than looking through one huge file.
